# OmniVerse NEWS

## Version 0.1.0

- Initial release.
- Shiny app for bulk RNA-seq and microarray analysis (GEO download, QC, normalization, DE, WGCNA, pathways, PPI, ML, optional immune deconvolution).
- All dependencies declared in DESCRIPTION; install via `BiocManager::install("OmniVerse")` (no runtime package installation).
